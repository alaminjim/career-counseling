Career Counseling Website

This is a career counseling website designed to provide professional guidance and resources to users seeking career advice. The website offers personalized career counseling services, resume review services, and much more. Built using React, React Router, Firebase for authentication, and hosted on [Netlify/Firebase/Surge], the platform is mobile-responsive and secure.

Live Site

https://candid-pie-e11d19.netlify.app/register

Features

- **User Authentication**: Password and Google-based login, email verification, and the ability to update user profile information.
- **Dynamic Content**: Displays personalized content for logged-in users and allows access to protected routes for service details and profile information.
- **Responsive Design**: The website is fully responsive and optimized for mobile, tablet, and desktop views.
- **Protected Routes**: Ensures that users cannot access service details or profile pages without being logged in.
- **Service Cards**: Users can view different services offered (e.g., Career Counseling, Resume Review) with pricing, counselor details, and more.
- **Comments/Feedback**: Users can leave comments or feedback for each service they engage with.
- **Dynamic Titles**: The website features dynamic titles for each page based on the content being viewed.
